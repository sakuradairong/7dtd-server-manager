# 支持说明

本项目是社区维护的开源工具，不提供商业 SLA。

## 获取帮助

请优先使用 GitHub Issues：

- Bug：请使用 Bug Report 模板，提供系统、版本、复现步骤和日志片段。
- 功能建议：请使用 Feature Request 模板，说明使用场景和预期收益。
- 安全问题：请按 `SECURITY.md` 处理，不要公开披露可利用细节。

## 提交问题前

请先确认：

1. 已搜索现有 issues。
2. 已使用最新版本或最新 `main` 分支复现。
3. 已阅读 `README.md` 中的开发、运行和安全提示。
4. 不包含服务器密码、公网 IP、令牌或其他敏感信息。
