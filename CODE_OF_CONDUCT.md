# 行为准则

本项目采用 Contributor Covenant 行为准则的简化版本。我们希望社区保持友善、专业和包容。

## 我们鼓励的行为

- 尊重不同经验水平、背景和观点。
- 使用建设性语言讨论问题和代码。
- 聚焦项目目标和技术事实。
- 在指出问题时尽量提供复现步骤、上下文或改进建议。

## 不可接受的行为

- 人身攻击、骚扰、歧视性语言或威胁。
- 泄露他人隐私信息。
- 故意发布恶意代码、凭据或有害链接。
- 持续偏离主题、刷屏或破坏项目协作。

## 执行方式

维护者可以对违反行为准则的内容进行隐藏、编辑、关闭、锁定，或限制相关账号参与本项目。

如果你需要报告行为准则问题，请通过仓库维护者在 GitHub 上公开提供的联系方式联系。请提供相关链接、截图或上下文，方便维护者判断。
